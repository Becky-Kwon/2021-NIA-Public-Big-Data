# 교육과정
- 빅데이터 분야 일경험을 희망하는 청년들을 대상으로 온라인 사전 교육 (1개월), 
- 빅데이터전문교육 (2개월)과 행정, 공공기관에서의 일경험수련(4개월/6개월) 기회를 제공하는 프로그램
- 최종적으로 전문교육이 끝나고 4개월간 **서울연구원**에서 연구인턴으로 근무함.
